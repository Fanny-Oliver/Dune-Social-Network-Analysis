# Dune-Social-Network-Analysis
⚠️ Spoiler Alert: This project contains analysis and insights that reveal character relationships, plot developments, and potential future storylines from Dune, Dune Messiah, and Children of Dune.

This project explores the character network of the Dune universe using Social Network Analysis (SNA). By extracting interactions from the text and applying network analysis techniques, it uncovers key characters, hidden relationships, and narrative structures across the series.

🚀 Project Overview

The objective of this project is to:

- Identify key characters and their influence
- Analyze interactions and relationships
- Detect communities and factions
- Generate insights into story progression and potential future dynamics

🧠 Methodology
- NLP (SpaCy) for Named Entity Recognition (NER)
- Manual mapping of 700+ character aliases
- Network construction using co-occurrence relationships
- Centrality analysis: Degree, Betweenness, Closeness
- Community detection (Louvain method)

🔍 Key Insights
- Paul Atreides emerges as the most influential character, acting as the central connector in the network
- Jessica, Stilgar, Alia, and Leto I play strong intermediary roles
- The Fremen dominate the interaction network, while Atreides remain central
- The narrative follows a hub-based structure, where few characters drive most interactions
- Identified 9 distinct communities, reflecting factions and alliances

🛠️ Tools & Technologies
- Python
- SpaCy
- Pandas
- NetworkX
- Gephi

💡 Why This Project Matters

- This project demonstrates how unstructured text can be transformed into meaningful network insights, helping uncover patterns in storytelling, influence, and character dynamics.
